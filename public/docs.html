Quick Start Guide

5 min read Updated June 23, 2025 Getting Started

Get up and running with ChatBot Builder Pro in just a few minutes. This guide will walk you through creating your first chatbot from start to finish.

  • Overview
  • Account Setup
  • Choosing a Builder

Overview

ChatBot Builder Pro provides three different ways to build chatbots, each designed for different skill levels and use cases. Whether you're a beginner looking for simplicity or an advanced user needing full control, we have the right tool for you.

New to chatbots? We recommend starting with the Simple Builder to get familiar with the concepts, then moving to more advanced builders as needed.

Account Setup

Before you can start building, you'll need to create an account:

  1. Click the "Get Started" button in the top navigation
  2. Fill out the registration form with your details
  3. Verify your email address
  4. Complete your profile setup

Once your account is set up, you'll be taken to your dashboard where you can manage all your chatbots.

Choosing a Builder

We offer three different builders to suit different needs:

Simple Builder

Perfect for beginners or quick prototypes. Features include:

  • Template-based approach
  • Minimal configuration required
  • One-click deployment
  • Basic customization options

Enhanced Designer

Great for users who want more design control. Features include:

  • Rich visual editor
  • Advanced styling options
  • Animation controls
  • Custom themes and components

Visual Builder

The most powerful option for complex chatbots. Features include:

  • Node-based visual programming
  • Advanced flow control
  • API integrations
  • Real-time collaboration
  • Version control
Pro Tip: You can switch between builders at any time. Start simple and upgrade your bot as your needs grow.

Building Your First Bot

Let's create a simple customer support chatbot using the Simple Builder:

Step 1: Create a New Project

1. Go to your dashboard
2. Click "Create New Chatbot"
3. Choose "Simple Builder"
4. Select "Customer Support" template

Step 2: Customize Your Bot

The template comes with pre-configured responses. You can:

  • Edit welcome messages
  • Add your company information
  • Configure contact details
  • Set up FAQ responses

Step 3: Configure Settings

Adjust these basic settings:

  • Bot Name: Give your bot a friendly name
  • Avatar: Upload a profile picture
  • Language: Set the primary language
  • Timezone: Configure for your location

Testing & Deployment

Before going live, always test your chatbot:

Built-in Testing

Use our integrated testing tool to:

  • Test conversation flows
  • Verify responses
  • Check integrations
  • Validate user experience

Deployment Options

Once you're satisfied with your bot, you can deploy it to:

  • Your Website: Embed as a widget
  • Messaging Platforms: Slack, Discord, WhatsApp, etc.
  • Social Media: Facebook Messenger, Instagram
  • Custom Applications: Via our API
Important: Always test your chatbot thoroughly before deploying to production environments.
